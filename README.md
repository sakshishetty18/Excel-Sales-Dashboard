# Excel Sales & Target Performance Dashboard

## Overview

Interactive Excel dashboard built to analyze sales performance, targets, orders, and category-wise contributions.

## Features

* KPI Cards
* Sales Trend Analysis
* Region-wise Analysis
* Product Category Analysis
* Interactive Slicers

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
